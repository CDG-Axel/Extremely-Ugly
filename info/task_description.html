Ugly numbers are numbers whose only prime factors are 2, 3 or 5. The sequence 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, ... shows the first 11 ugly numbers. By convention, 1 is included. Write a program to find and print the N’th ugly number. Same as previous task with a minor difference - input value limit is 1000000.

Input: N, int.

Output: N'th Ugly Number, int.

Example:

{% if is_js %} 
ugly_number(4) == 4
ugly_number(6) == 6
ugly_number(11) == 15
ugly_number(999) == 51018336
{% else %} 
ugly_number(4) == 4
ugly_number(6) == 6
ugly_number(11) == 15
ugly_number(999) == 51018336
{% endif %}

How it’s used: Demonstrate that straight way can be very slow

Precondition: Given int is below or equal of 1000000
